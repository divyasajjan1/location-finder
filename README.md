# Location Finder (Landmark Recognition)

A local, end-to-end project that:
- Trains a CNN to recognize famous landmarks from images
- Predicts the landmark from an uploaded image
- Enriches results with location, distance, and AI-generated summaries

Tech stack:
- PyTorch
- Django REST Framework / FastAPI
- DuckDuckGo (ddgs)
- BeautifulSoup
- Geo APIs
- OpenAI API
